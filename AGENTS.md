# AGENTS.md

## Project style
This project is not a generic diagram-rendering task. It is a visual-philosophical cartography.

## Priorities
1. Preserve conceptual structure.
2. Enhance aesthetic intensity.
3. Prefer expressive composition over sterile neatness.
4. Keep outputs reproducible.
5. Document structural vs stylistic changes separately.

## Visual direction
Constructivist, cybernetic, poetic, diagrammatic, elegant, tense, layered.

## Avoid
- corporate infographic aesthetics
- excessive simplification
- default Graphviz look
- childish color palettes
